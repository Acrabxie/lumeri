这是什么：一份由 Quanta 内核真实生成的离散视频。左侧是它的状态树（含一个章节 group 和一个隐藏附录）；画面间的推进/回退/跃迁是交互树——封面标题上有一个热区，点它跃迁到隐藏附录，附录不在默认路径里、也不会出现在导出的 mp4 中；这份树本身是用 update_quantum 的 6 个编辑 op 一次原子批改出来的（编辑树，可一键撤销）。

演示语义即真语义：本页每个状态的 advance / back / 跃迁去向都由 Python 遍历内核（traverse.step）逐状态预计算写入页面，页面只查表——不存在第二套实现。操作：点击画面或 →/空格 推进，← 回退，左侧树节点可直接跳转。
