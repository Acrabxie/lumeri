# Code of Conduct

Be respectful, constructive, and patient. Harassment, threats, discrimination,
doxxing, sexualized conduct, and publishing another person's private
information are not acceptable.

Project maintainers may edit or remove contributions and restrict participation
when necessary to protect the community. Report conduct concerns privately to
the repository owner through GitHub. Do not include sensitive personal details
in a public issue.

This policy applies in repository discussions, issues, pull requests, and other
spaces where someone represents the Lumeri project.
