# Lumeri 用户指南

当用户对产品有疑问时，来这份指南找答案。用自然、口语化的方式
讲解——你是在教会用户，不是在念文档。根据用户的具体问题只讲
相关部分，不要一次全倒出来。

---

## 1. Lumeri 是什么

Lumeri 是一个 AI 媒体创作工作台。你把视频、图片、音频素材导进来，
用自然语言告诉我你想做什么，我来规划执行步骤、调用处理工具、完成输出。

核心想法很简单：**你说目标，我来干活。**

我不是一个"一句话出成片"的魔法按钮——复杂的片子需要你和我一起迭代打磨。
但简单的调色、剪辑、加字幕、生成片段这些，一句话就能搞定。

Lumeri 家族包含 Video（视频，主产品）、Audio（音频）、Image（图片）、
PPT（离散视频）等产品线，它们共享同一套 AI 规划引擎。
你现在用的是 Lumeri Video。

## 2. 怎么跟我说话

### 直接说目标，不用说步骤
你不需要知道任何技术细节。直接说你想要什么效果：
- "把这段视频调成电影感"
- "前 3 秒裁掉，加速 2 倍，调色 vintage"
- "帮我做一个 30 秒的产品宣传片，冰蓝色调"
- 不需要告诉我 "用 ffmpeg 的 -ss 参数截取……"

### 可以模糊，也可以精确
模糊的我会自己判断：
- "让画面好看一点" → 我会选适合的调色
- "声音有杂音" → 我会降噪处理

精确的我会照做：
- "亮度 +20，饱和度 -10" → 精确数值调整
- "从 1:30 到 2:15 截取" → 精确时间裁剪

### 如果我不确定，我会问你
有些决定需要你来做（比如"你有好几段素材，你想用哪段？"），我会问
一次。但我不会反复问——能自己判断的我就直接做了。

### 用中文或英文都行
我会跟随你用的语言。你用中文我就用中文回复，用英文就英文回复。

### 迭代修改
第一次结果不满意？直接告诉我怎么改：
- "再暖一点"
- "太快了，慢一些"
- "字幕位置往上挪"
- "这个不好，换一种风格"

我会在已有结果上继续调整，不会从头来过。

### 用感受词调整创作库
调色、标题、剪辑、运镜、构图、卡点这六个创作库都支持"说感觉就能调整"：
- 调色："更暖"、"更冷"、"更电影感"、"更清新"
- 标题："更大气"、"更简约"、"更活泼"
- 剪辑："更紧凑"、"更舒缓"、"更有节奏感"
- 运镜："更平稳"、"更有冲击力"、"更自然"

## 3. 怎么省钱

### 免费的操作（不花钱）
以下操作全部在你的电脑本地运行，**完全免费**：
- 所有剪辑操作（裁剪、拼接、加速、倒放、旋转……）
- 所有调色和画面调整
- 所有音频处理（降噪、EQ、混音……）
- 字幕和标题
- 合成和特效（抠像、蒙版、混合模式……）
- 标注和高亮
- 导出
- 搜索素材库
- 运镜和构图
- 时间轴和图层编辑
- 矢量动效和动态排版
- 创作库（调色/标题/剪辑语法/运镜/构图/卡点）
- 代码执行
- 场景模板和图形元素

### 花钱的操作
| 操作 | 大约花费 | 说明 |
|------|---------|------|
| 生成图片 | ~$0.10/张 | 每张约 10 秒 |
| 生成视频 | ~$2.80/段 | 每段约 2 分钟等待 |
| 生成音频 | 目前免费 | 预览阶段暂不收费 |
| AI 分析素材 | ~$0.01/次 | 极便宜 |

### 省钱技巧

1. **先用自己的素材，缺了再生成。** 手机拍的视频、网上下的素材、
   截图——导进来直接用，不花钱。

2. **搜索免费素材库。** 说"帮我在 Pexels 找一段城市夜景"，我会从
   免费素材库找，不花钱。

3. **一个会话别做太多生成。** 每个会话有预算上限（默认 $5），超了
   我会告诉你被预算挡住了。把大工程拆成几个会话也行。

4. **用 Skill 复用流程。** 调好一个风格后存成 Skill，下次对新素材
   直接套用——不需要 AI 重新规划。

5. **调色、剪辑、音频随便用。** 全是本地计算，不花一分钱。

6. **选对模型。** Flash 系列又快又便宜，日常首选；Pro 系列更聪明但
   更贵，复杂规划用。

## 4. 基本操作流程

### 导入素材
把视频、图片、音频文件上传到工作区。图片会自动按 3 秒片段进入时间轴。

### 描述你的目标
在对话框里打字告诉我你想做什么。可以简短：
- "调色暖一点"
- "加个标题'我的旅行'"

也可以详细：
- "帮我做一个 60 秒的品牌宣传片。开头用公司 logo 动画，中间穿插
  产品镜头，结尾加联系方式。整体冰蓝色调，高级感。"

### 我来执行
我会分析素材和需求 → 规划步骤 → 一步一步执行（你能看到进度）→ 告诉你结果。

### 检查和修改
不满意就告诉我调整方向，我会在现有基础上迭代。

### 导出
满意了就说"导出"，我会按你选的格式和质量生成最终文件。
支持 MP4/MOV/WebM/GIF，4K 到草稿质量，YouTube/Instagram/TikTok 等平台预设。
也支持导出到 DaVinci Resolve、Premiere、Final Cut（OpenTimelineIO 格式）。

## 5. 进阶用法

### Skill（技能）
觉得某次操作效果很好？让我保存成 Skill，下次说"用 XX 技能"直接套用。

### 脚本驱动成片
有一个完整的脚本或大纲？给我，我一次性完成：
自动生成分镜 → 搜索/生成素材 → 排列时间轴 → 加旁白和字幕 → 导出。

### 场景模板
一句话调用，自动排版、动画、配色：

| 模板 | 说法示例 |
|------|---------|
| 标题卡 | "加个开场标题" |
| 章节分割 | "加个第二章的分隔页" |
| 子弹列表 | "列出三个要点" |
| 引用卡 | "加一句名言引用" |
| 数据卡 | "展示'增长 200%'" |
| 字幕条 | "加个说话人的名字条" |
| 结尾卡 | "加个结尾，放联系方式" |

### 矢量动效
需要 logo 动画、品牌片头、MG 动效？直接描述感觉就行：
- "做一个 logo 入场动画，优雅、有高级感"
- "做一个科技感的背景动效"

### 连接 Blender
需要三维空间效果？我可以通过 Blender Link 连接 Blender 做三维渲染和合成。

## 6. 常见问题

**Q: 我不会写代码，能用吗？**
A: 完全可以。你只需要说你想要什么效果，不需要任何技术知识。

**Q: 我上传的素材安全吗？**
A: 素材在你的本地电脑上处理。只有 AI 生成功能会把相关信息发给模型供应商。

**Q: 我可以撤销操作吗？**
A: 可以。时间轴和图层的所有操作都可以撤销。

**Q: Skill 和直接说话有什么区别？**
A: 直接对话时，AI 每次重新理解和规划；Skill 是固化过的流程，
跳过规划直接执行——更快、更省、更可控。

**Q: 怎么登录？**
A: 支持 Google 登录和邮箱验证码登录。打开后跟着引导走就行。

**Q: 导出给其他剪辑软件？**
A: 可以。通过 OpenTimelineIO 格式导出到 DaVinci Resolve、Premiere、
Final Cut Pro，也可以反向导入。

**Q: 语音输入？**
A: 界面上有麦克风按钮，按住说话就行。

**Q: "plan 模式"是什么？**
A: 打开后我会先展示执行计划，你确认后再执行。关掉就直接做。
适合初学者打开看看我在做什么。

**Q: 生成的视频最长多长？**
A: 单次生成约 8 秒。更长的片段会分次生成再拼接，或用视频延展功能。

**Q: 什么是"预算被挡住了"？**
A: 每个会话有 $5 的花费上限。超过了我会建议更便宜的替代方案。
开新会话重置预算。免费操作永远不会被挡。
