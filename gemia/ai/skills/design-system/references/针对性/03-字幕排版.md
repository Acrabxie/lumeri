# 针对性指导手册 · 字幕排版(Caption Typography)

> 引用 [`DESIGN.md`](../DESIGN.md) 的 `{typography.*}` / `{frame.*}` / `{motion.text-*}` /
> `{colors.*}` token,**不重复数值**。字幕排版决定成片的"精致度",规则朴素但必须守。

## 1. 字体:少而有层级

- **家族 ≤ 2**:标题用 `{typography.title}`、正文/字幕用 `{typography.caption-body}`,
  最多再加一种衬线做强调。同帧禁止 3 种以上字体或字重。
- **字号有梯度**:标题 > 正文 `{typography.caption-body}` > 标注 `{typography.caption-annotation}`;
  层级靠字号+字重区分,而非堆颜色。
- **字体气质匹配题材**:题材与字体性格要对(蛮荒题材别用圆润卡通体)。

## 2. 位置:贴底、不居中、不盖脸

- **贴下方安全区**:垂直位置距底边 `{frame.caption-bottom-margin}`、左右留 `{frame.caption-side-margin}`;
  工业级字幕永远在下方,居中糊在画面中央是无主次意识的表现。
- **不遮关键区**:字幕不得盖住人脸或画面重要信息;标题不得盖住风景主体。
- **在安全框内**:文本整体落在 `{frame.title-safe}` 内,防止不同平台裁切丢字。

## 3. 可读性:描边与对比

- 字幕与背景对比度 ≥ 4.5:1(WCAG AA);不足时用 `{colors.caption-stroke}` 描边或加底衬。
- 浅底用 `{colors.text-on-light}`、深底用 `{colors.text-on-dark}`;避免亮字压亮景导致糊读。

## 4. 时长:跟阅读速度走

- 显示时长 = 字数 ÷ 阅读速度(中文约 5–7 字/秒),不短于 0.3s、不长于该镜内容合理时长。
- 一屏字数有上限(横屏一行约 15–20 字),超了拆两行或拆两屏,别缩字号硬塞。

## 5. 动画:出入必有缓动

- 进场用 `{motion.text-enter}`、出场用 `{motion.text-exit}`(ease-out 入 / ease-in 出);
- 禁止 0→1 生硬跳转(像数字时钟);透明度/位移变化过渡 ≥ 10 帧。

## 6. 竖屏与多语言

- **竖屏 `{frame.aspect-vertical}` 重排**:竖屏字幕上移(避开底部平台 UI 遮挡区)、行更短、字号相对更大。
- **多语言**:不同语种字长差异大,布局按最长语种预留;CJK 与拉丁字体分别指定,避免混排跳动。

## 7. 位置决策树(简版)

```
是标题/片头? → 用 {typography.title},上/中区,时长 1.5–3s
是主字幕/对白? → 用 {typography.caption-body},下方安全区,时长跟阅读速度
是标注/角标? → 用 {typography.caption-annotation},贴近被标注元素,弱化不抢戏
```

## 8. Gemini 自检 prompt(植入字幕工作流)

```
字幕审查(逐条):
1. 全片字体家族是否 ≤2?同帧字体/字重是否 ≤2?
2. 字幕是否在下方安全区、未居中、未盖脸?是否在 {frame.title-safe} 内?
3. 与背景对比度是否 ≥4.5:1?否则加 {colors.caption-stroke} 描边。
4. 出入是否用 {motion.text-enter}/{motion.text-exit} 缓动?
5. 显示时长是否匹配字数×阅读速度(≥0.3s)?竖屏是否已重排上移?
命中任一 → 修正并复检。
```
