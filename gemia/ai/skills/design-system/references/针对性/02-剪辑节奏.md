# 针对性指导手册 · 剪辑节奏(Editing & Pacing)

> 引用 [`DESIGN.md`](../DESIGN.md) 的 `{pacing.*}` / `{transition.*}` / `{audiosync.*}` /
> `{components.*}` token,**不重复数值**。这是"从业余跨到专业"的**最大杠杆(P0)**。

## 1. 节奏:剪辑的灵魂

- **ASL 随情绪走**:基线 `{pacing.asl-default}`,按 `{pacing.asl-by-mood}` 因段调整
  (访谈慢、叙述中、高能快、沉静更慢)。
- **时长要有分布**:按 `{pacing.duration-distribution}` 铺出长/中/短/超短,制造山谷与山峰。
  全片恒定 ASL = 打卡式剪辑,是机器感的头号来源。
- **禁平坦**:触发 `{pacing.flatness-guard}`(≥4 连续同时长)即判"节奏平坦",插长/超短镜打破。
- **高能不过载**:快切段 ≤ `{pacing.high-energy-run-max}`;高能后必须给缓冲长镜"呼吸"。

## 2. 切点技法(切在对的瞬间)

- **Cut on action**:切在动作**幅度最大**的瞬间,注意力随动作走,观众感觉不到切。
- **Match cut**:靠形状/颜色/运动方向匹配衔接(弱/中/强三级),强化主题或对称。
- **J-cut**(声先入 1–3s):制造预期,为新镜头预告,减轻切换生硬。
- **L-cut**(画切声留):制造余韵与过渡,保持连贯。
- 特写单镜 ≤ `{components.shot-dialogue.duration}` 量级参考,过长显诡异(见主册 `{pacing.cu-max}`)。

## 3. 空间逻辑(别让观众迷路)

- **180° 轴线**:相机保持在轴线同侧,人物左右位置稳定;越轴需用过渡宽镜或正面镜"重置"。
- **视线匹配**:A 看向画外 → 切到其所视之物,方向要对得上。
- **信息递进**:建立镜头(宽,2–4s)→ 中景 → 特写;别从特写直切建立镜头(倒序、混乱)。

## 4. 结构技法

- **蒙太奇**:多个 2–3s 短镜压缩时空,音乐驱动、通常无对白;段落 10–30s 为宜。
- **并列/交叉剪辑**:两条线交替,切换频率**递增**以堆紧张。

## 5. 音乐与呼吸

- **卡点纪律**:约 `{pacing.beat-snap-ratio}` 的切点故意强踩拍制造高能,其余服从叙述自然;逐拍全卡 = 剪辑秀、失叙述。
- **对齐容差**:剪辑点踩拍 ≤ `{audiosync.beat-snap-tolerance-ms}`;关键点(章节/高潮/收尾)必须踩拍。
- **留白**:长镜头 3–8s = 沉静,10s+ = 等待/不适;长镜头前后要有快慢对比,否则拖沓。

## 6. 转场纪律(Less is more)

- 默认直切 `{transition.default}`;全片转场 ≤ 镜头数 × `{transition.budget-ratio}`。
- 每个转场须有理由(踩拍/段落切换/时空大跳);禁止连续两转场、单镜多转场、特效炫技。

## 7. Gemini 自检 prompt(植入剪辑工作流)

```
剪辑审查(整段):
1. 读音乐 BPM/拍点;快板段配短镜、慢板段配长镜,切点落在拍点 ±{audiosync.beat-snap-tolerance-ms}。
2. 统计已剪镜头时长序列 → 是否符合 {pacing.duration-distribution}?是否触发 {pacing.flatness-guard}?
3. 逐切点标注技法(cut-on-action / match / J / L)与理由;无理由的快切删除。
4. 检查越轴与视线匹配;信息是否按建立→中景→特写递进。
5. 转场计数 ≤ 镜头数 × {transition.budget-ratio}?超则删到只剩战略转场。
命中任一 → 重剪并说明修改点。
```
