# 针对性指导手册 · 调色(Color & Grade)

> 引用 [`DESIGN.md`](../DESIGN.md) 的 `{grading.*}` / `{colors.*}` token,**不重复数值**。
> 对标 DaVinci Resolve 调色台水准。核心心法:**先技术、后风格;克制;连贯。**

## 1. 流程:一级 → 二级(顺序不可颠倒)

**一级校色(技术地基)**——先把画面"摆正":
1. **曝光**:调 Lift/Gain,让肤色中间调落在 `{grading.midtone-skin-target}`;黑点不低于
   `{grading.black-point-floor}`、白点不高于 `{grading.white-point-ceiling}`。
2. **白平衡**:用 Kelvin 值校正(而非色相/饱和滑块——那会连带扭曲其他色彩),以中性灰或肤色为参考。
   白平衡错了,后面风格化再华丽也救不回。
3. **对比**:服从内容——访谈偏平、戏剧偏硬。

**二级校色(风格与情绪)**——在正确的地基上做风格,见第 3 节。

## 2. 肤色:调色的生死线

- 矢量示波器上肤色点须**贴肤色线**(红→黄象限对角线);脱线就是"绿脸/火星人"。
- 肤色饱和在全局基础上再压 `{grading.skin-saturation-offset}`;与背景同饱和 = 显假。
- 脸部高光(前额、颧骨)必须留细节,不能过曝或糊成一坨。
- **偏绿**=绿幕溢出/白平衡偏绿 → 用 HSL/Keyer 隔离修;**偏洋红**=欠曝后用色温硬补 → 先补曝光再微调。

## 3. Lift/Gamma/Gain 与情绪

- **分层逻辑**:先用 **Gamma** 定整体亮度感受(眼睛最敏感)→ 用 **Lift** 保暗部细节 → 最后 **Gain** 微调高光。
- **成套情绪基调**(直接引用 preset,不要手搓):
  - `{grading.presets.cinematic}` 电影暖调 · `{grading.presets.documentary}` 纪实中性
  - `{grading.presets.faded-film}` 褪色胶片 · `{grading.presets.noir}` 冷硬暗黑
- **冷暖分区**:全片定 1–2 个主色温,只在明确转折点切换(最多 2–3 次),切换处成段过渡;
  让高光/中间调/暗部**独立游走**,而非一个色温压死全片。

## 4. 饱和克制(反廉价核心)

- 全局不超过 `{grading.saturation-global-cap}`;>0.85 起即"滤镜/贴纸感"。
- 分色域差异化:肤色压、环境略提、强调色(灯光)可多提;矢量点云半径 ≤ 刻度 60–70%。

## 5. LUT:是起点不是终点

- 套 LUT 后仍需按 `{grading.lut-microtune}` 幅度微调(曝光/对比/肤色)适配本镜。
- LUT 强度用 `{grading.lut-strength}` 而非拉满,保留后续灵活性;禁止多个 LUT 盲目叠加。
- **反 teal-orange 惯性**:优先按情绪(冷/暖/褪色/胶片)选,而非套"青蓝+橙皮肤"的缩略图公式。

## 6. 示波器三剑客(判定,不靠肉眼)

1. **波形图**:查曝光——左端不贴底、右端不贴顶(无死黑爆白)。
2. **矢量示波器**:查肤色在线、查饱和半径。
3. **直方图**:查曝光分布——覆盖是否够宽,是否偏曝/欠曝。

## 7. Gemini 自检 prompt(植入调色工作流)

```
调色审查(逐镜):
1. 肤色点是否在肤色线上?饱和是否已按 {grading.skin-saturation-offset} 压过?
2. 波形两端是否触及 {grading.black-point-floor} / {grading.white-point-ceiling} 边界?
3. 全局饱和是否 ≤ {grading.saturation-global-cap}?
4. 本镜与相邻镜色温差是否 ≤ {grading.cross-shot-temp-tolerance}?
5. 情绪基调是否取自 {grading.presets.*} 而非手搓 teal-orange?
命中任一 → 标记并回修,附示波器读数证据。
```
