# Security Policy

## Reporting a vulnerability

Please use GitHub's private vulnerability reporting for this repository:

https://github.com/Acrabxie/lumeri/security/advisories/new

Do not open a public issue containing exploit details, credentials, private
media, local filesystem paths, or personal information.

Include the affected revision, reproduction steps, expected impact, and any
safe proof-of-concept material. You should receive an acknowledgement within
seven days.

## Supported versions

Security fixes are applied to the current `main` branch. Preview releases may
change quickly and should not be exposed directly to the public internet.

## Public/private boundary

This repository contains the local creative runtime. Hosted authentication,
email delivery, cloud account management, billing, and subscription systems
are intentionally outside the public codebase.
